# Flight Status

Simple react project to show flight status based on the api

1. https://flight-status-mock.core.travelopia.cloud/flights
2. https://flight-status-mock.core.travelopia.cloud/flights/:id

## Installation

`npm install`

## Run locally

`npm run dev`

## Run test

`npm run test`

## Build

`npm run build`
